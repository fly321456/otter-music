### v2.0.1 发版说明

🚀 **新功能与核心特性**

- **智能音源匹配升级**：在设置中新增“智能音源”开关，播放付费/VIP歌曲时自动搜索并切换至免费音源；手动或自动匹配成功后，现在会直接将对应歌单/收藏中的原受限歌曲替换为免费版本。
- **网易云功能深度整合**：
- **市场拓展**：歌单市场新增「我的」分类，登录后可直接查看网易云的“每日推荐”、“创建的歌单”与“收藏的歌单”。
- **全新搜索**：接入并支持网易云官方搜索接口。
- **详情跳转**：支持从单曲菜单直接跳转查看对应的「歌手详情页」和「专辑详情页」。
- **快速分享**：网易云歌单等详情页右上角新增“分享”按钮（复制链接）。
- **全局错误页**：新增友好的应用路由错误与 404 兜底页面。

✨ **优化与改进**

- **登录重构**：重写网易云扫码登录逻辑，引入原生二维码生成机制，解决二维码加载失败及轮询失效问题。
- **匹配算法**：提升完整版歌曲的搜索匹配精准度（现必须满足歌名与歌手名称双重严格一致）。
- **UI 与适配**：
- 全面适配移动端底部安全区（Safe Area），彻底解决全面屏经典手势下底部 Tab 与悬浮播放栏相互遮挡的问题。
- 「我的」页面自定义歌单列表现已支持展示歌单封面。
- 移除冗余的付费/试听状态标签，优化视觉层级。
- **性能优化**：对网易云 API 引入分级的 TTL 缓存策略（短/中/长时），大幅降低冗余网络请求；列表强制启用虚拟化渲染，操作更流畅。

🐛 **问题修复**

- 修复歌词解析时部分时间戳正则匹配不生效的问题。
- 修复因跨端环境导致本地缓存音频文件链接失效的问题（失效后自动回退网络播放）。
