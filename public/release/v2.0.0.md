# Otter Music v2.0.0 🦦

全新 2.0 版本，聚焦**发现**与**体验**。新增歌单市场，重构底层播放逻辑，移动端体验全面升级。

## 🎉 新特性

- **歌单市场**
  新增瀑布流式歌单探索页，支持按语种、风格、场景筛选，遇到喜欢的歌单可**一键导入**。
- **智能音源匹配**
  试听片段自动识别，菜单新增「解锁完整音源」功能，一键匹配全网免费高音质版本，告别 30 秒限制。
- **歌单封面自定义**
  支持自定义本地歌单封面，可上传网络图片或直接提取首个歌曲封面。

## ✨ 体验升级

- **全局底部播放栏**
  重构 UI 交互，增大触控区域，处于子页面时固定在底部。
- **设置页重组**
  功能重划分为「个人偏好」、「账号数据」、「关于系统」三类，配置更高效。
- **动态歌曲标签**
  网易云歌单列表新增 VIP（金）、付费（红）、试听（灰）动态角标，权益状态一目了然。

## 🐛 修复与优化

- **网络层重构**：移动端通过 CapacitorHttp 绕过 CORS 限制，彻底解决封面加载失败及播放异常。
- **歌词解析升级**：优化正则解析逻辑，完美支持多时间戳复杂歌词，杜绝漏句。
- **扫码登录优化**：增加请求时间戳防止缓存，解决手机确认后状态偶发不跳转问题。
