# OtterMusic v2.0.3

## ✨ 新特性

- **沉浸式播放器**：全屏播放界面新增基于专辑封面的「动态主题色提取」背景，并针对移动端性能和 UI 视觉做了安全钳制与多层质感优化。
- **CI 自动构建发布**：新增 Android GitHub Actions 工作流，现在推送 `v*` Tag 即可自动完成打包、签名并发布 APK。
- **Android 返回键适配**：全面接管移动端物理返回键，支持优雅地关闭弹窗、退出全屏播放器或最小化应用。

## 🚀 体验优化

- **UI/UX 升级**：
- 全新优化的网易云（歌单/歌手/专辑）详情页，新增骨架屏（Skeleton）过渡，改善列表及海报视觉排版。
- 播放器进度条新增悬停/拖拽光效指示器（Dot），交互更清晰。
- 微调全局暗色模式（Dark Mode）基础色板，背景更加深邃。

## 🐛 问题修复

- 修复歌单广场上拉加载更多时，可能导致数据列表出现重复项的问题。
- 修复在移动端菜单对部分网易云歌曲点击“查看歌手/专辑”时，因缺失 ID 导致无法准确跳转的问题（现已加入自动补全机制）。

## 🛠 工程化与测试

- 补充了针对 API 降级策略、历史记录和播放队列管理的单元测试（Vitest）。
- 代码库引入 Gradle Wrapper，彻底解决本地与 CI 构建环境的不一致问题。
- 重构了 ESLint 规则配置（Flat Config）。
