# OtterMusic v2.3.0

feat:

- 新增「边听边缓存」功能，播放时自动缓存，可离线播放，节省流量
- 新增「通过歌单链接导入」功能：支持网易云、酷狗、酷我、咪咕、QQ音乐歌单
- 新增「下载设置管理」：支持下载音质选择、内嵌封面/歌词、自定义下载目录（仅移动端APP）
- 支持选择音源 Badge 是否常驻显示
- 新增「全屏背景模式选择」功能，支持深色背景、封面遮罩和主题色提取三种模式
- 新增网页端管理后台功能，方便自建用户配置 SYNC_KEY 做数据同步
- 添加 PWA 支持，桌面端、移动端（iOS 请使用 Safari）都支持安装。
- 支持「自建后端」功能，用户可在设置中配置 API 地址接入自部署的后端服务，实现数据同步。

其他：

- 修复单曲循环时音频链接过期问题
- 客户端与服务端共享逻辑迁移至 @otter-music/shared
- 修复优化了部分已知问题
